# react-native-sendbird-calls

React Native Sendbird Calls SDK

## Installation

```sh
npm install react-native-sendbird-calls
```

## Usage

```js
import { multiply } from "react-native-sendbird-calls";

// ...

const result = await multiply(3, 7);
```
